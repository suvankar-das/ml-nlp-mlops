Set UP
============================================

Create an environment > command

> conda create -p venv python==3.10

After creating environment , write the below command

> conda activate [Environment_name_without_[]]

If You want to install any packages , 
    - First make sure if you have entered correct package name inside package.txt

Then , run below command,     
> pip install -r package.txt


In order to run , You will need 2 packages , 

> pip install ipykernel
